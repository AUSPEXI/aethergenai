AML Graph Typology Library: Parameterised Patterns for Testing

Fraud detection demands realistic behaviours. AethergenPlatform provides a library of parameterised AML typologies to generate test graphs and evaluate detection—safely and repeatably, without real PII.

Typologies

Using the Library
