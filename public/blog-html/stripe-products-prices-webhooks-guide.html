Stripe Setup: Products, Prices, Webhooks, and Environment Variables

This guide shows how we configure billing for AethergenPlatform: creating Products and Prices, wiring secure webhooks, and mapping entitlements. Our approach keeps billing deterministic and audit‑ready.

What You Need

  • Products/Prices (recurring and one‑off)
  • Test/Live API keys
  • Webhook signing secret

Steps

  1. Create Products and Prices in Stripe.
  2. Set env vars: STRIPE_SECRET_KEY, STRIPE_WEBHOOK_SECRET.
  3. Implement checkout and portal links.
  4. Handle webhook events: checkout.session.completed, invoice.paid, subscription.updated.

AethergenPlatform Entitlements

Our create‑checkout‑session function maps purchased SKUs to platform entitlements (e.g., generation quotas, evidence export). The stripe‑webhook function updates access and writes an evidence log for each state change—so finance and audit can trace outcomes.

Why This Matters

Predictable, auditable billing is part of buyer trust. By treating billing as code—in the same pipeline as packaging and evidence—we keep operations aligned and verifiable.
